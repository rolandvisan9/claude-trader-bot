# Proyecto
Este proyecto es un bot de trading en fase inicial.

# Reglas importantes
- Usar solo Alpaca paper trading.
- No usar live trading.
- No usar margen.
- No usar cripto.
- No ejecutar órdenes reales sin validación explícita.
- Priorizar seguridad, logs y simplicidad.
- Explicar todo paso a paso porque el usuario tiene nivel bajo de Python.

# Estado actual
El sistema ya puede:
1. Leer cuenta y posiciones.
2. Leer una watchlist.
3. Leer precios recientes de mercado.
4. Construir un daily brief consolidado.

# Objetivo actual
Construir primero una capa de observación y análisis.
Todavía no se deben enviar órdenes.
