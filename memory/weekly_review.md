# Revisión semanal

## Formato
Cada viernes o lunes, añadir una entrada con:
- Semana del (fecha)
- Resumen de P&L de la semana
- Trades realizados (referencia al trade_log)
- Patrones observados
- Ajustes a la estrategia o reglas de riesgo
- Objetivos para la próxima semana
