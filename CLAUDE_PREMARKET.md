# Rutina Pre-Market

## Qué hacer
1. Leer CLAUDE.md para contexto del proyecto.
2. Verificar que estamos en Alpaca paper trading.
3. Ejecutar:
   - python scripts/get_portfolio_snapshot.py
   - python scripts/get_market_snapshot.py
   - python scripts/fetch_news_headlines.py
   - python scripts/build_daily_brief.py
   - python scripts/generate_daily_analysis.py
   - python scripts/generate_smart_analysis.py
   - python scripts/detect_unusual_movements.py
4. Leer memory/daily_brief.md y memory/smart_analysis.md.
5. Generar un resumen ejecutivo de 5-8 líneas.
6. **NO ejecutar órdenes ni conectar live trading**.
7. Guardar el resumen en memory/premarket_summary.md.
8. Confirmar que todos los archivos se actualizaron correctamente.
